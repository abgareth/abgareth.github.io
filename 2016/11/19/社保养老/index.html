一、社会养老保险分为哪3类？

社会基本养老保险，是国家根据法律、法规的规定，强制建立和实施的一种社会保险制度。在这一制度下，用人单位和劳动者必须依法缴纳养老保险费，在劳动者达到国家规定的退休年龄或因其他原因而退出劳动岗位后，社会保险经办机构依法向其支付养老金等待遇，从而保障其基本生活。

根据目前国家最新的政策，社会基本养老保险分为3大类：“城镇职工基本养老保险”、“城乡居民养老保险”、“机关事业单位养老保险”。在实际中，其实还有“退伍军人养老保险”这个针对特殊人群的养老保险，在这里就不展开，因为不同的服役地区有不同的待遇，比如在西藏服役就和在西宁服役的养老待遇就不一样。

二、您属于哪一类？

《中华人民共和国社会保险法》第十条中规定：

职工应当参加基本养老保险，由用人单位和职工共同缴纳基本养老保险费。

无雇主的个体工商户、未在用人单位参加基本养老保险的非全日制从业人员以及其他灵活就业人员可以参加基本养老保险，由个人缴纳基本养老保险费。

公务员和参照公务员法管理的工作人员养老保险的办法由国务院规定。

这项法规体现了三类群体，“职工”、“灵活就业人员”、“公务员”。但由于我国存在一种户口的奇葩制度，所以在实际中，还会针对城镇户口、农村户口有所区分。按照参保对象划分，这三类群体可选则的基本养老保险如图所示：

由上图可以看到，城镇户口可选择的范围最广，三种基本养老保险你都有资格参保，而农村户口只能参保城乡居民基本养老保险。这里重点提一下，“城乡居民基本养老保险”的前身是“新型农村社会养老保险”（简称新农保）和城镇居民社会养老保险（简称城居保）。2014年2月7日，国务院总理李克强主持召开国务院常务会议，会议决定合并新型农村社会养老保险（简称新农保）和城镇居民社会养老保险（简称城居保），建立全国统一的城乡居民基本养老保险制度。

如果你是企业职工，同时也是城镇户口，那你就是自（qiang）动（zhi）参加城镇职工基本养老保险：

“企业职工”+“城镇户口”=“城镇职工基本养老保险”

如果你是灵活就业人员（比如初创者、个体户、集市摊贩、保险代理人等），同时也是城镇户口，那你可以参保两种社会养老保险：

“灵活就业人员”+“城镇户口”=“城镇职工基本养老保险（以个人身份参保）” 或 “灵活就业人员”+“城镇户口”=“城乡居民基本养老保险”

如果你是公务员，同时也是城镇户口，那你也是自（qiang）动（zhi）参加机关事业单位养老保险：

“公务员”+“城镇户口”=“机关事业单位养老保险”

如果你是农村户口，无论是何种职业，都只能参加“城乡居民基本养老保险”。不过别担心，这其实是最具性价比的社会养老保险，可以说是，谁不买，谁吃亏。后文会详细讲解。

“不限职业”+“农村户口”=“城乡居民基本养老保险”

三、每类社会养老保险该如何缴费参保及所享受的养老待遇

##1.城乡居民基本养老保险 中国国务院总理李克强2014年2月7日主持召开国务院常务会议，听取关于2013年全国人大代表建议和全国政协委员提案办理工作汇报，决定合并新型农村社会养老保险和城镇居民社会养老保险，建立全国统一的城乡居民基本养老保险制度。

这是大部分老百姓最容易忽略但实际上性价比最高的社会养老保险，相当划算，可以说谁不买谁吃亏。它是完全由个人参保缴纳，退休时由国家财政出资负责的，回报相当给力。

在此以重庆市为例（因为我在重庆）详细解读。 ###如何缴费参保 城乡居民养老保险的参保缴费是完全由个人缴纳费用，每年缴纳100~2000元不等，分为12个档次，每个档次对应的政府补贴不同，最少缴费15年。如下表： ！[城乡缴费标准]

养老待遇-死亡待遇

领保人员死亡后，从死亡的次月起停发基本养老金，其亲属或利益相关人应在30日内将有效死亡证明提交给参保地社会保障服务所，并按规定领取一次性死亡补助金，标准为死亡时本人上月养老待遇乘以12个月。并退还其个人账户余额，包括个人缴费、政府补贴、集体补助3个部分。

死亡补助金 = 12 × 每月养老金 退还余额 = 个人账户余额 = 个人缴费 + 政府补贴 + 社会集体补贴和个人资助 + 上述金额产生的利息

养老待遇-月退休养老金

城乡居民社会养老保险待遇由基础养老金和个人账户养老金组成，支付终身。

月养老金 = 基础养老金 + 个人账户总额 ÷ 139 基础养老金 = 95（今后随国家政策调整），且缴费满15年后每多缴1年基础养老金增加2元 个人账户总额 = 个人缴费 + 政府补贴 + 社会集体补贴和个人资助 + 上述金额产生的利息 注：社会集体补贴和个人资助是指，有条件的村集体经济组织，应当对参保人给予补助，补助标准由村委员会召开村民会议民主确定，鼓励有条件的社区将集体补助纳入社区公益事业资金筹集范围。鼓励其他社会经济组织、公益慈善组织、个人为参保人员缴费提供资助。集体补助每年最高不得超过2000元。虽然政府鼓励集体补助，但这仍只是自愿行为。

举例说明月基本养老金的计算过程: : 以2015年25岁的小明为例，他每年缴纳100元（对应的政府补贴的30元）参加城乡居民基本养老保险，连续缴费15年，并假设基础养老金95元一直不变（当然不可能，国家肯定会因为通货膨胀而逐年提升）。到了60岁退休时，他能领取的月基本养老金为：

基础养老金 = 95（缴费满15年后每多缴1年基础养老金增加2元） 个人账户养老金 = 100×15 + 30×15 + 0 + 0 = 1950（忽略社会补贴和利息） 月基本养老金 = 95 + 1950 ÷ 139 = 109 : 即，小明60岁退休时，每月能够拿到109元的养老金，一年就是1308元。14个月时间就能拿回自己15年所缴纳的1500元本金。 : 如果小明连续缴纳35年，一直缴到60岁，根据计算，他那时能够拿到95+20×2+130×35÷139=168元的月养老金，一年就是2016元，20个月的时间就能拿回自己35年所缴纳的3500本金。而且这还是忽略了个人账户的利息，利率遵循银行的最新一年期存款利率并按照年复利计息。所以实际情况下，几乎是不到一年就拿回本金了。 : 以此类推可以算出：小明以最高档2000元缴纳15年的月基本养老金为95+2140×15÷139=326元，一年3912元。需要92个月才能拿回自己缴纳的30000元本金。小明以最高档2000元缴纳35年的月基本养老金为95+20×2+2140×35÷139=674元，一年8086元。需要103个月才能拿回自己缴纳的70000元本金。

所以从以上的计算过程中可以看到，按照最低档次100/年的缴纳，虽然每月只能领取109元养老金，但实际上却是回本最快的方式，14个月即可回本。随后就是活多久领多久了。性价比绝对超值！强烈建议还没有社会养老保险的朋友购买！

##2.城镇职工基本养老保险（企业参保） 这是大部分老百姓所参保的养老保险，参保缴费也很简单，几乎不用自己去操心。这也让很多人忽略了自己的利益。 它是由用人单位统一向其纳税所在地的社会保险经办机构申报办理职工参保缴费，缴费比例为28%，其中单位承担20%，职工个人承担8%，最少缴费15年。这是全国统一的缴费参保标准。这里先用一张不完全正确的图解解释下具体的缴费和退休后的养老待遇。尽管这张图解有一些误导，但理论思路是没错的，有助于大家理解。后文会详细解释。 保险] 图中的缴费参保是假设开始缴纳社保时（25岁）个人工资为5000，假设退休时当地上一年（60岁）的社会平均工资依然为5000。看起来非常简单，但其实它是这三类社会养老保险中最复杂的一个。根据参加工作的时间不同，退休的时间不同，个人的工资收入不同，所享受的待遇也是不同。各个地方的政策略有不同，但总体思路是一样的。

在此以重庆市为例（因为我在重庆）详细解读。

###如何缴费参保 尽管我们都知道，缴费比例为28%，其中单位承担20%，职工个人承担8%。单位承担的20%计入统筹账户，归国家所有；职工承担的8%计入个人账户，归个人所有。但是比例的基数是什么？是什么的20%？什么的8%？这里就要先解释一下**缴费基数和社平工资**两个概念.

缴费基数并不是一个固定值，它是一个参考值，是和个人月平均工资进行对比所得。缴费基数有上限和下限的限定。最高不能超过上限，最低不能低于下限。

缴费基数上限 = 社平工资 × 300% 缴费基数下限 = 社平工资 × 60%

社平工资：上年度全市城镇非私营单位职工月平均工资。这是一个每年都会变动的数据，逐年递增。

举个栗子来说明，2016年的参保**缴费基数**即是以重庆市2015年年社平工资62091来计算。

2015年社平月均工资 = 62091÷12 = 5174.25 缴费基数上限 = 5174.25 × 300% = 15523 缴费基数下限 = 5174.25 × 60% = 3105

好，这样就有了我们需要关注的两个数据，缴费基数上限和缴费基数下限，我们就能知道20%、8%到底是多少了。

如果您的月均工资大于每年的缴费基数上限，以上限为缴费基准； 如果您的月均工资小于每年的缴费基数下限，以下限为缴费记住； 如果您的月均工资在上限与下限之间，以您的月工资为缴费基准。 比如2016年的缴费基数： : 您2015年的月均工资大于15523（缴费基数上限），以15523为2016年缴费基准； 您2015年的月均工资小于3105（缴费基数下限），以3105为2016年缴费基准； 您2015年的月均工资大于3105小于15523（介于上限与下限之间），以您2015的月均工资为2016年缴费基准。

###养老待遇-死亡待遇 城镇职工基本养老保险能够提供两项待遇，死亡待遇和养老待遇。

死亡待遇 = 丧葬费 + 一次性救济金 + 遗嘱生活困难补助

1．丧葬补助费 　　丧葬补助费标准由各省、自治区、直辖市确定。中央国家机关及其所属事业单位执行所在地的标准。目前重庆市为2000元。 2．一次性抚恤金 　　一次性抚恤金的数额，根据职工死亡性质确定，因病死亡的，按职工病故时的10个月的工资额发给：因公牺牲的，按牺牲时20个月的工资额发给；批准为革命烈士的，按牺牲时的40个月的工资额发给。 　　一次性抚恤金，国家机关职工死亡后，由其家属户口所在地的民政部门发给，事业单位职工死亡后，由其原工作单位发给。 3．遗属生活困难补助 　　职工死亡后，遗属生活有困难的，死者生前所在单位，可以根据“困难大的多补助，困难小的少补助，不困难的不补助”的原则，给予定期或临时补助。遗属生活困难补助标准，一般以能维持当地群众生活水平为原则，具体标准由各省、自治区、直辖市规定。中央国家机关及其所属事业单位，执行所在地区的标准。 　　遗属补助费按应享受遗属补助的人数和标准计算，其总额不得超过死者生前的工资。

从收益的角度来说，如果按照退休前个人月工资5000来测算，新增的丧葬费和一次性救济金并不多，大概2-3万，可能刚好办个葬礼。

###养老待遇-月退休养老金 这恐怕是每个人最关心的地方了，可以说，很大程度上，它决定了我们晚年的生活品质，决定了我们晚年喝粥还是吃肉。但由于国家政策调整、个人参加工作的时间不同等原因，月退休养老金针对不同的人群，其计算方式也不同。以重庆为例，大致分为以下三种情况（其他地方可能由于政策不同，实行政策的时间有所不同）：

年1月1日及其之后参加工作（或参加基本养老保险），2011年1月1日及其以后退休或符合按月领取基本养老金条件的人员，基本养老金由基础养老金和个人账户养老金组成。

月基本养老金 ＝ 基础养老金＋个人账户养老金

基础养老金 = （A+A×Q）÷2×M×1%

个人账户养老金 = 个人账户（含利息） ÷ 养老金计发月数

A：退休时上年度全市城镇非私营单位职工月平均工资。 Q：平均缴费指数。即，自己实际的缴费基数与社会平均工资之比的历年平均值。通俗的解释就是，您每月工资除以社会平均工资的值，再将这缴费15年（甚至更多年）的所有值平均化。平均缴费基数介于0.6与3之间。

举例说明平均缴费指数: : 假设2015年的社平月均工资5174，2015年缴费基数上限为5174×3=15523，缴费基数下限为5174×0.6=3105。再假设2015年您每月工资都是5000，介于3105与15523之间，那您的缴费基数就是你的月均工资5000。那么5000÷5174=0.9663就是您2015年的平均缴费基数。 假设2016年的社平月均工资5667，2016年缴费基数上限为5667×3%=17000，缴费基数下限为5667×0.6=3400。再假设2016年您每月工资都是8000，介于3400与17000之间，那您的缴费基数就是您的月均工资8000。那么8000÷5667=1.441就是您2016年的平均缴费基数。

M：累计缴费年限。 计发月数：退休时的年龄不同，计发月数也不同。

举例说明月基本养老金的计算过程: : 以2015年25岁的小明为例，现在年薪9万，2015年社平工资为7.6万。则小明的平均缴费基数Q为9÷7.6=1.18。假设小明每年工资都不变，59岁年薪依然是9万，月薪7500。并且假设社会平均工资每年也不变，那么小明59岁时社平工资依然为为7.6万，月均6333元，且平均缴费基数Q依然是7500÷6333=1.18。

: 小明每年缴纳8%的收入计入个人养老金账户，总计为7500×8%×12×35=252000。

: 由此，可得出缴纳35年的养老保险后，退休时的养老金为：

基础养老金 = （6000+6000×1.25）÷2×35×1% = 2362.5 个人账户养老金 = 252000÷139 = 1813 职业年金 = (126000+252000)÷139 = 2719 月基本养老金 = 2362.5+1813+2719 = 6894.5 : 即，小明60岁退休时，每月能够拿到17369的养老金。对比59岁时退休前的月薪7500，养老金替代率为4229÷7500 = 56.38%。那小明要获多久才能把自己之前缴纳的个人养老金252000拿回来呢？252000÷4229=59.58个月，即4.96年，65岁的时候。之后就白领国家的钱了。

####情况二、大明-1996年1月1日及其之前工作（2011年之后退休） 1996年1月1日及其之前参加工作（或参加基本养老保险），2011年1月1日及其以后退休或符合按月领取基本养老金条件的人员，基本养老金由基础养老金、个人账户养老金、过渡性养老金组成：

月基本养老金＝基础养老金＋个人账户养老金 + 过渡性养老金

基础养老金 = （A+A×Q）÷2×M×1%

个人账户养老金 = 个人账户（含利息） ÷ 养老金计发月数

过渡性养老金 = （A+A×Q）÷2×N×1.4%

A：退休时上年度全市城镇非私营单位职工月平均工资。 Q：平均缴费指数。即，自己实际的缴费基数与社会平均工资之比的历年平均值。通俗的解释就是，您每月工资除以社会平均工资的值，再将这缴费15年（甚至更多年）的所有值平均化。平均缴费基数介于0.6与3之间。 M：累计缴费年限。 计发月数：退休时的年龄不同，计发月数也不同。 N：本人建立个人养老金账户之前的缴费年限。

举例说明月基本养老金的计算过程: : 以1990年25岁的大明为例，现在年薪9万，1990年社平工资为7.6万。则小明的平均缴费基数Q为9÷7.6=1.18。假设大明每年工资都不变，59岁年薪依然是9万，月薪7500。并且假设社会平均工资每年也不变，那么大明59岁时社平工资依然为为7.6万，月均6333元，且平均缴费基数Q依然是7500÷6333=1.18。在1996年建立个人账户之前N为6年，1996年建立个人账户之后到2025年60岁退休时，个人账户累计缴费29年。

: 大明每年缴纳8%的收入计入个人养老金账户，总计为7500×8%×12×29=208800。

: 由此，可得出缴纳35年的养老保险后，2026年退休时的养老金为：

基础养老金 = （6333+6333×1.18）÷2×35×1% = 2416 个人账户养老金 = 208800÷139 = 1502 过渡性养老金 = （6333+6333×1.18）÷2×6×1.4% =580 月基本养老金 = 2416+1502+580 = 4498 : 即，大明60岁退休时，每月能够拿到4498的养老金。对比59岁时退休前的月薪7500，养老金替代率为4498÷7500 = 59.97%。那大明要获多久才能把自己之前缴纳的个人养老金208800拿回来呢？208800÷4498=46.42个月，即3.86年，64岁的时候。之后就白领国家的钱了。

####情况三、老明-1996年1月1日及其之前工作（2011年之前退休） 1996年1月1日及其之前参加工作（或参加基本养老保险），2011年1月1日及其以前退休或符合按月领取基本养老金条件的人员，基本养老金由基础养老金、个人账户养老金、过渡性养老金、调节性（又称补贴性）养老金组成：

月基本养老金＝基础养老金＋个人账户养老金 + 过渡性养老金 + 调节性养老金

基础养老金 = A×20%

个人账户养老金 = 个人账户（含利息）÷120

过渡性养老金 = A×Q×N×1.4%

调节性养老金 = 70×（N÷M）

A：2010年全市城镇非私营单位职工月平均工资。 Q：平均缴费指数。即，自己实际的缴费基数与社会平均工资之比的历年平均值。通俗的解释就是，您每月工资除以社会平均工资的值，再将这缴费15年（甚至更多年）的所有值平均化。平均缴费基数介于0.6与3之间。 M：累计缴费年限。 计发月数：退休时的年龄不同，计发月数也不同。 N：本人建立个人养老金账户之前的缴费年限。

举例说明月基本养老金的计算过程: : 以1973年25岁的老明为例，假设年薪依然为9万，1973年社平工资为7.6万。假设老明每年工资都不变，59岁年薪依然是9万，月薪7500。并且假设社会平均工资每年也不变，那么老明59岁时社平工资依然为为7.6万，月均6333元，且平均缴费基数Q依然是7500÷6333=1.18。1973年25岁，到2008年60岁退休，总缴费35年。其中到1996年，建立个人养老金账户之前的缴费年限N为23年，1996年建立个人账户之后到2008年60岁退休时，个人账户累计缴费12年。

: 老明每年缴纳8%的收入计入个人养老金账户，总计为7500×8%×12×12=86400。 假设2010年的社平工资依然为7.6万，月均6333元。

: 由此，可得出缴纳35年的养老保险后，2016年的月基本养老金为：

基础养老金 = 6333×20% = 1267 个人账户养老金 = 86400÷120 = 720 过渡性养老金 = 6333×1.18×23×1.4% =2406 调节性养老金 = 70×（23÷35） = 46 月基本养老金 = 1267+720+2406+46 = 4439 : 即，大明60岁退休时，每月能够拿到4439的养老金。对比59岁时退休前的月薪7500，养老金替代率为4439÷7500 = 59.18%。那大明要获多久才能把自己之前缴纳的个人养老金65.03万拿回来呢？86400÷4439=19.46个月，即1.62年，62岁的时候。之后就白领国家的钱了。

这里有一个全球通用的重要指标，养老金替代率，它反映了退休人员的生活水平，也反映了一个国家或地区的社会福利待遇。但从以上测算的结果来看，替代率只有56%左右，这是一个比较差的指标，意味着我们退休之后的生活质量只有工作时的一半左右。并且这个测算指标还忽略了个人工资及社平工资的增长和通货膨胀，实际结果会更低。世界银行组织建议，要维持退休前的生活水平不下降，养老金替代率必需不低于70%，国际劳工组织建议养老金替代率最低标准为55%。并且这里忽略了单位缴纳的20%的统筹养老账户，如果您认为工资的20%上交国家进入统筹账户实际上是变相掠夺您的财产的话，要想靠单纯的每月基本养老金拿回来您之前缴的钱，那您就得多多保重身体，祈祷活个八九十岁。

另一方面，从测算的结果来看，想要在退休后拿回个人部分缴纳的8%。小明需要5年，大明需要4年，老明需要2年。而从目前人均寿命75岁来说，老明白领的钱最多最划算，小明白领的钱最少最吃亏。目前社会养老金制度采用现收现付制，即小明这类青年人群所缴纳的养老金立即通过国家转手付给了老明这类老年人群。随着人口出生率的下降，老龄化人口越来越多，国家的负担也会越来越重，这也是为什么有些地方的养老金严重亏空的原因。

最后，关于如何理解单位承担的20%一直有很多争议。如果换个角度来看，20%其实也是我们自己缴纳的，只不过是税前列支了。假设你月薪税前月薪10000，实际到手工资扣除可能只有6800多。。。。
